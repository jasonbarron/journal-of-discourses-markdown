# Journal of Discourses (1854-1886)

A complete markdown collection of the Journal of Discourses from The Church of Jesus Christ of Latter-day Saints, comprising volumes 1-26 (1854-1886).

## About

The Journal of Discourses is a 26-volume collection of public sermons and teachings by early leaders of The Church of Jesus Christ of Latter-day Saints. These discourses were delivered primarily in Salt Lake City and other locations throughout the Utah Territory during the latter half of the 19th century.

This repository provides the complete collection in accessible markdown format, enabling easy searching, referencing, and research.

## Repository Contents

### Complete Collection
- **`Journal_of_Discourses_Complete.md`** - All 26 volumes compiled into a single markdown file (32+ MB)

### Individual Volumes
The `individual-volumes/` directory contains all 26 volumes as separate markdown files:
- `JoD01.md` through `JoD26.md`

Each volume file includes the complete text of all discourses from that volume.

### PDF Sources
The `pdf-sources/` directory contains the original PDF files for all 26 volumes, preserving the source material for reference and verification.

### Extraction Summaries
The `extraction-summaries/` directory contains curated collections of discourses:
- **`Brigham_Young_Discourses.md`** - Compilation of all discourses by Brigham Young across the 26 volumes

## Usage

### Accessing the Complete Collection
For comprehensive searching and research across all volumes:
```bash
# View the complete collection
cat Journal_of_Discourses_Complete.md
```

### Accessing Individual Volumes
For targeted research within a specific time period or volume:
```bash
# View a specific volume
cat individual-volumes/JoD01.md
```

### Searching the Content
Use standard text search tools to find specific topics, names, or phrases:
```bash
# Search for a term across all volumes
grep -r "search term" individual-volumes/

# Search within the complete collection
grep "search term" Journal_of_Discourses_Complete.md
```

## Historical Context

These discourses were originally recorded by skilled stenographers and published in serial form. Although not doctrine, they represent an important historical record of early Latter-day Saint thought and pioneer life in the American West during the 19th century.
